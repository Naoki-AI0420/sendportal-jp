# sendportal-jp ビジネスプラン

## コンセプト
「メルマガ配信を月額9,800円で。Mailchimpの1/3の価格。」

## ターゲット
- 中小企業のマーケ担当（従業員10-100人）
- 今 Mailchimp/SendGrid に月3-5万払ってる層
- 「安くて日本語で使えるメルマガツール」を探してる人

## プラン構成

| プラン | 月額 | 購読者上限 | メール送信/月 |
|---|---|---|---|
| スターター | ¥4,980 | 1,000人 | 10,000通 |
| スタンダード | ¥9,800 | 5,000人 | 50,000通 |

## 技術スタック

### SaaS基盤
- sendportal（MIT）+ 日本語化済み
- PHP 8.2 + Laravel 10 + MySQL + Redis
- Docker Compose で顧客ごとにインスタンス or マルチテナント

### 決済
- Stripe Checkout（サブスクリプション）
- Stripe Customer Portal（プラン変更・解約）
- Stripe Webhook（支払い成功 → インスタンス自動作成）

### フロント（LP + ダッシュボード）
- LP: Next.js or Astro（高速・SEO最適化）
- 管理: sendportal のUI（日本語化済み）

## 販売導線

```
SEO/広告 → LP → Stripe Checkout → 自動プロビジョニング → sendportal利用開始
                                          ↓
                               VPSにインスタンス自動作成
                               DNS設定（サブドメイン）
                               SES設定（送信ドメイン認証）
```

## 自動プロビジョニング（最重要）

Stripe Webhook で checkout.session.completed を受けたら：
1. 顧客用のサブドメイン作成（customer1.merumaga.jp）
2. Docker コンテナ起動（sendportal インスタンス）
3. 初期設定（管理者アカウント、SES設定）
4. ウェルカムメール送信
5. 利用開始
